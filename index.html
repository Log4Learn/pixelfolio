自己紹介

フリーランスのWebエンジニアとして5年以上の経験があります。中小企業向けの業務システム開発やポータルサイト構築、マッチングサイトの開発などを手がけてきました。要件定義から保守運用まで一貫して対応可能です。

スキル

PHP / 各種フレームワーク

Laravelを中心に、CakePHP、FuelPHP、CodeIgniter、Zend Frameworkなどの実務経験あり。ポータルサイトや業務管理システムの構築に対応。

Java / Play Framework / C#

Play Frameworkでの業務アプリケーション開発や、C#によるデスクトップ向け業務ツール開発に携わった経験あり。

Ruby / Python

RubyやPythonを用いたスクリプト作成、簡易データ処理、業務自動化ツールなどを開発した経験あり。API連携やCSV処理などに活用。

MySQL / データベース設計

複数のプロジェクトにおいて、正規化と拡張性を考慮したスキーマ設計、SQLパフォーマンス調整、マイグレーション管理を担当。

JavaScript / Vue.js / React / Angular / Alpine.js

フロントエンド開発ではVue.jsをメインに、ReactやAngularを使用したページ構築・画面実装経験もあり。Alpine.jsは静的サイトでの簡易インタラクションに活用。

Git / GitHub / GitLab / AWS / Docker

Gitの基本操作、GitHub/GitLabによるリポジトリ管理。AWS（SSH接続、ログ調査、デプロイ）、Dockerでの開発環境構築にも対応。

Linux / Shell / Apache / Nginx

Linux環境での開発、cronによるスケジューリング、Apache/Nginxの構成編集やデプロイ設定の実務経験あり。

Kintone API / Slack API

Kintone APIによる業務アプリの拡張開発、Slack APIを活用した通知連携・自動投稿などの社内業務効率化ツールを作成。
